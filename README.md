# zhenxun_plugin_guessname
 真寻的音游猜字符插件

# 命令使用

### 开字符 [数量]

创建新游戏

默认10题

范围为1~题库上限

题库内歌名重复自动省略

此题库为phigros旧版数据 可以自行在song.data(文本方式)中添加

如:``开字符 15``

### k/打开 [关键词]

开出被隐藏的字符

关键词数量长度不限,以空格分隔

如:

``k f``

``k a e i o u``

``打开 eg``

### 验证/c

``验证 [歌名] [歌曲编号]``

或

``c[歌曲编号].[歌名]``

猜出对应歌曲的歌名

如:

``验证 rrhar'il 5``

``c5.rrhar'il``

### 结束

不想玩了？

游戏发起者可以在任意时间内发送``##结束``来显示答案

其他人可以在游戏开始后{总题数}分钟来发生``##结束``显示答案
